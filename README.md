# Face Mask Detection using CNN and Deep Learning

## 📌 Project Overview
This project is a Deep Learning-based Face Mask Detection system developed using Python and TensorFlow/Keras. The model detects whether a person is wearing a face mask or not through image classification techniques using Convolutional Neural Networks (CNN).

The project was developed as part of an internship on **Deep Learning using Python** at Global Techno Solutions, Chennai.

---

## 🚀 Features
- Face Mask / No Mask Classification
- CNN-based Deep Learning Model
- Image Data Preprocessing
- Data Augmentation
- Model Training and Testing
- Accuracy and Loss Visualization
- Real-time Prediction Ready

---

## 🛠 Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- OpenCV
- CNN (Convolutional Neural Network)

---

## 📂 Dataset
- Dataset collected from Kaggle
- Around 12,000 images
- Two classes:
  - Mask
  - No Mask

---

## 🔄 Workflow

1. Data Collection
2. Data Preprocessing
3. Image Augmentation
4. CNN Model Building
5. Model Training
6. Model Evaluation
7. Prediction and Visualization

## 🧠 CNN Architecture
The CNN model includes:
- Convolution Layers
- Pooling Layers
- Flatten Layer
- Dense Layers
- Dropout Layer
- Sigmoid Activation Function
